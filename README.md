**Автор:** Забродин И.Н.

## Типы файлов

| Расширение | Описание |
|---|---|
| `.cir` | Схема Microcap / SPICE-нетлист |
| `.bak_cir` | Резервная копия схемы |
| `.cir.inx` | Индексный файл Microcap |
| `.DNO` | Данные анализа (Microcap output) |
| `.TNO` | Результаты transient-анализа |
| `.pdf` / `.docx` | Отчёты по лабораторным работам |
| `.zip` | Архив схем и сопутствующих файлов |

---

## Прикладные части

### Лабораторные работы (`lab1` – `lab7`)
Каждая папка содержит схемотехническое моделирование конкретной задачи в Multisim/Microcap. Папки `*def` содержат вариант задания или защитную версию работы.

### Рубежные контроли (`rk`, `rk2`)
Файлы промежуточных контрольных работ (схемы и/или расчёты).

### Отчёты (корень репозитория)
PDF и DOCX отчёты по каждой лабораторной, включая ВАХ диода (`VAX_dioda`) и отчёты ЛР 1–7, упакованные в `.zip`-архивы.

### Вспомогательное (`mc12cd`)
Компонентная база или настройки среды Microcap 12.

---

## Инструменты

- **Microcap 12** — аналоговое SPICE-моделирование (`.cir`, `.DNO`, `.TNO`)
- **Multisim** — схемотехническое моделирование
- **Python** — дополнительные расчётные задачи
